# Docify

Docify is an intelligent OCR-based document processing system that converts images into searchable text and provides full-text search functionality.

## Features
- OCR text extraction using Tesseract
- Store extracted text in SQLite database
- Full-text search system
- Simple desktop GUI using Tkinter

## Tech Stack
- Python
- Tkinter
- SQLite
- PyTesseract

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   python app.py

## Project Purpose
Built as a learning project to understand OCR, database systems, and desktop application development.
