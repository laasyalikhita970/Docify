# Docify

Docify is an OCR-powered intelligent document search system built using Python, Tkinter, SQLite, and PyTesseract.

It converts non-machine-readable documents into searchable digital text and enables fast full-text document retrieval.

---

## Features

- OCR text extraction from images
- PDF text extraction support
- SQLite database storage
- Full-text search engine
- Keyword-based document retrieval
- Desktop GUI using Tkinter

---

## Tech Stack

- Python
- Tkinter
- SQLite
- PyTesseract
- pdfplumber
